## 单词的反转

#### 1.题目

'abc as'--------------->>>>>>>'as abc'

#### 2.分析

两次反转

#### 3.代码

```python
def func(s, start, end):
    s_len = len(s)
    i, j = start, end
    while i < j:
        tmp = s[i]
        s[i] = s[j]
        s[j] = tmp
        i += 1
        j -= 1
def reverseStr(s):
    s = list(s)
    func(s, 0, len(s)-1)
    start = 0
    for i in range(len(s)):
        if s[i] == ' ':
            func(s, start, i-1)
            start = i+1
    print(s)
    print(''.join(s))
reverseStr("fds sfsdf fa")
```
