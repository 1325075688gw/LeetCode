## 异或操作

#### 1.性质

- 异或操作满足`交换律`和`结合律`
- **0和一个数异或，等于该数**
- 两数相同，异或，等于0
- 两数不同，异或，等于其他数

#### 2.1找出只出现一次的元素

给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

**说明：**

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

**示例 1:**

```python
输入: [2,2,1]
输出: 1
```

**示例 2:**

```python
输入: [4,1,2,1,2]
输出: 4
```

#### 代码

```python
    # 异或解决
    def singleNumber(self, nums):
        a = 0
        for num in nums:
            a = a ^ num
        return a
```

#### 2.3找出出现两次的数

```python
def findDup(nums):
    nums_len = len(nums)
    i = 0
    for j in nums:
        i ^= j
#     print(i)
    j = 1
    while j < nums_len:
        i ^= j
        j += 1
    print(i)
```

#### 2.4`异或`来判断一个二进制数中1的数量是奇数还是偶数

- 求10100001中1的数量是奇数还是偶数； 答案：1 ^ 0 ^ 1 ^ 0 ^ 0 ^ 0 ^ 0 ^ 1 = 1,结果为1就是奇数个1，结果为0就是偶数个1；

```python
res = 0
for i in nums:
	res = res ^ i
```
